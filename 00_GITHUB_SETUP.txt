## Troubleshooting GitHub Actions

**Q: Build failed on Windows**
A: Likely ncurses issue. The workflow has `vcpkg` for Windows deps.

**Q: No artifacts uploading**
A: Check Actions tab for build errors. Fix locally with `make` first.

**Q: How to trigger rebuild?**
A: Push an empty commit:
```bash
git commit --allow-empty -m "rebuild"
git push
```

**Q: Can I build for ARM (Pi, Moto)?**
A: Yes! Add to build.yml:
```yaml
- runs-on: ubuntu-latest
  with:
    arm: true
```
